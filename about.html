About This Project

Built with the MERN Stack (Vanilla JS Frontend)

Task Management System is a full-stack web application designed to help users organize their daily workflow seamlessly.

Tech Stack

  • Frontend: HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB via Mongoose

Key Features

  • Secure User Authentication
  • Real-time CRUD operations for tasks
  • Responsive, interactive Web3-style UI
  • Local Storage session management
